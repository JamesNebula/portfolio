Project Overview

A full-stack Flask application that helps forex traders make data-driven decisions by combining multi-timeframe technical analysis with machine learning predictions.

Confluencer is live and I use it regularly. It assists in: reducing emotional trading, quantifying confluence across timeframes, and providing objective probability forecasts. The app features secure user authentication, persistent trade tracking, and model retraining capabilities.

Confluence Scoring System

Weighted analysis combining 5 key factors:

  • 30% Timeframe alignment (M5 to D1)
  • 25% Pattern confirmation (candlestick patterns)
  • 20% Support/Resistance proximity
  • 15% EMA alignment (9, 21, 50, 200)
  • 10% Volume confirmation
